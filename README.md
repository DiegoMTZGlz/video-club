# Video Club - Práctica

Práctica video-club

### Pre-requisítos

node-js

### Ejecución

Usar:

```
npm start
```

o crear la imagen de Docker:

```
docker build -t videoclub .
```

para despues ejecutarla con:

```
sudo docker run -dti -p80:80 videoclub
```
o acceder a:

  - [Back4App](https://videoclub-olxlme8c.b4a.run/)
